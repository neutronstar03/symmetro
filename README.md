# symmetro
Encrypt and decrypt text with AES-256-GCM on a easy to deploy webpage

## Usage
- have node 20.x or higher installed
- install pnpm (`corepack use pnpm@latest`)
- `pnpm install`
- `pnpm start` (for dev mode)
- `pnpm build` (for production build)
